# Telegram Auto Shop Bot 🤖

بوت تيليجرام متكامل لإدارة الطلبات الرقمية واستلام إثباتات الدفع ومراجعتها من قبل الأدمن بشكل مباشر داخل تيليجرام.

تم تطوير البوت باستخدام Python و python-telegram-bot مع دعم كامل لإدارة الطلبات وحجز الكميات ومتابعة المدفوعات.

---

# ✨ المميزات

* استقبال طلبات المستخدمين بشكل تلقائي.
* عرض المخزون المتاح بشكل مباشر.
* نظام حجز للكميات المطلوبة.
* مؤقت Countdown للطلبات.
* استقبال صور إثبات الدفع.
* إرسال إثبات الدفع مباشرة إلى الأدمن.
* أزرار قبول ورفض للطلبات.
* إدارة المستخدمين المحظورين.
* أوامر حظر وفك حظر للمستخدمين.
* حفظ الطلبات تلقائياً.
* حفظ بيانات المستخدمين.
* حفظ قائمة المحظورين.
* دعم عدة عناوين دفع للعملات الرقمية.
* منع إنشاء أكثر من طلب نشط لنفس المستخدم.
* إشعارات فورية للمستخدم والأدمن.

# ⚙️ نظام الطلبات

1. يبدأ المستخدم البوت.
2. يتم عرض المخزون والأسعار.
3. يحدد المستخدم الكمية المطلوبة.
4. يتم حجز الكمية مؤقتاً.
5. يظهر عداد زمني لإتمام الدفع.
6. يرسل المستخدم صورة إثبات الدفع.
7. يتم تحويل الطلب للأدمن للمراجعة.
8. يقوم الأدمن بالموافقة أو الرفض.
9. يتم تحديث حالة الطلب تلقائياً.

# 👨‍💻 أوامر الأدمن

### حظر مستخدم

```text

```

```text

```
